Elite:Dangerous Data Collection - Web Service



Usage:
Type of serviceFunctionWeb Service URLReturn type
Solar Systems
Search by name /solarsystem/byname?name=<name> List of solar systems
Search by ID /solarsystem/<id> The select solar system or empty
Get Stations /solarsystem/<id>/stations Return a list of stations
Creation (GET) /solarsystem/create?name=<name> Created solar system with ID
Creation (POST, JSON) /solarsystem/create
method=POST, Content-Type: application/json
{"name":String,"x":Long,"y":Long,"z":Long}		 Created solar system with ID
Update (PUT, JSON) /solarsystem/update
method=POST, Content-Type: application/json
{"idSolarSystem":Long,"name":String,"x":Long,"y":Long,"z":Long}		 The updated solar system
Stations
Search by name /station/byname?name=<name> List of stations
Search by ID /station/<id> The selected station or empty
Get market goods /station/<id>/market List of goods in the station's market
Get buyable ships /station/<id>/shipyard List of buyable ships in the station
Get buyable ships /station/shipyard?idstation=<id> List of buyable ships in the station
Creation (GET, SolarSystem ID required) /station/create?id=<solarsystemid> Created station with ID
Creation (POST,JSON) /station/create
method=POST, Content-Type: application/json
{"idSolarSystem","name","isBlackMarket","isMarket","isShipyard","isOutfitting"}		 Created station with ID
Update (GET) /station/update?id=<id>&name=<name>
   &market=<false/true> &blackMarket=<false/true>
  &shipyard=<false/true> &outfitting=<false/true>		 The updated station
Update (PUT,JSON) /station/update
method=PUT, Content-Type: application/json
{"idStation":Long,"idSolarSystem":Long,"name":String,"isBlackMarket":boolean,
"isMarket":boolean,"isShipyard":boolean,"isOutfitting":boolean}		 The updated station
Add a buyable ship /station/addship?idstation=<id>&idship=<id> nothing
Remove a buyable ship /station/removeship?idstation=<id>&idship=<id> nothing
Ship Outfit Categories
Get all categories /ship/outfitcategories List of outfit's categories
Get all categories' name /ship/outfitcategories/names List of names (String)
Buyable Ships
Get all buyable ships /ship/buyables List of buyable ships
Get a specific ship /ship/buyables/<id> The selected ship
Get a ship outfit's slots /ship/buyables/<id>/slots List of outfit's slots
Goods
Get all goods' categories /goods/categories List of goods' categories
Get all goods' designations (sub-categories) /goods/designations List of goods' designations
Get a specific goods (create it if empty) /goods/get?station=<id>&designation=<id> One specific goods
Get all goods in a station's market
(See also /station/<id>/market)		 /goods/station?id=<id> List of goods in the station's market
Update a goods (GET, IDs required) /goods/update?station=<id>&designation=<id>&price=<price>
   &number=<number>&supplyOrDemand=<0/1/2>&priority=<0/1/2/3>
attributes: 0=reset value, supply=1/demand=2, priority high=1/med=2/low=3)		 nothing
Update goods (PUT,JSON) /goods/update
method=PUT, Content-Type: application/json
Body= List of Goods
[{"idStation":Long, "idDesignation":Long, "price":Int,
   "number":Long, "supplyOrDemand":Int, "priority":Int},{...}]		 nothing
